# 3D Solar System

Interactive 3D solar system visualization built with vanilla WebGL and JavaScript. Features realistic planetary orbits, textured planets, Saturn's rings, a pulsing sun, and a star-field skybox.

**[Live Demo](https://zebiv.com/solar_system/)**

## Controls

- Drag to rotate the camera
- Scroll to zoom in/out
- Touch supported on mobile

## Tech

- WebGL (no libraries)
- Vanilla HTML/CSS/JavaScript
- GLSL shaders for planet lighting, sun glow, rings, and skybox
- Planet textures from Solar System Scope
